`pdp parse`
---

> CSV Parsing Utilities

You can parse the CSV file using the following commands:

```bash
pdp parse --headers-only
```

**FLAGS:**

Name         | Param            | Required | Default   | Definition
-------------|------------------|----------|-----------|----------------------------------------------
Headers Only | `--headers-only` | No       | `false`   | Extract CSV Headers to File
