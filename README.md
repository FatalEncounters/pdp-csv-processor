Install
---

#### Requirements

- [X] [Node v11+](https://nodejs.org/en/download/)

#### Install via NPM

```bash
npm install -g pdp-csv-processor
pdp fetch
```

#### Install via Clone

```bash
cd ~
git clone https://github.com/FatalEncounters/pdp-csv-processor.git
cd pdp-csv-processor
npm install -g
pdp fetch
```
